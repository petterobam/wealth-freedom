# v1.7.0 数据可视化大屏

> **版本目标**：全屏沉浸式财务数据展示，适合演示、截图、文章配图
> **预计工时**：12-16h（3-4次心跳）

## 核心功能

### 1. 大屏入口与布局
- 全屏模式入口（侧边栏 + Dashboard 按钮）
- 响应式网格布局（CSS Grid）
- 深色主题（默认，适合大屏展示）

### 2. 数据模块（6大卡片）
| 模块 | 内容 | 图表 |
|------|------|------|
| 净资产总览 | 总资产/总负债/净资产 | 数字翻牌器 |
| 收支趋势 | 近12月收入/支出 | 双折线图 |
| 资产配置 | 各账户占比 | 环形图 |
| 预算执行 | 各类别预算vs实际 | 水平柱状图 |
| 储蓄进度 | 三阶段目标进度 | 多段进度条 |
| 投资收益 | 持仓盈亏 | 热力色块 |

### 3. 交互功能
- 数据实时刷新（与桌面端数据同步）
- 截图导出（一键保存为PNG）
- 自动轮播模式（每5秒切换高亮模块）

## 文件清单


## 技术方案
- 图表：复用现有 ECharts（桌面端已有）
- 布局：CSS Grid 3x2
- 数据：复用现有 database.ts API
- 路由：/bigscreen，全屏时隐藏侧边栏
