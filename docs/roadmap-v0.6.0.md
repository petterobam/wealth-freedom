# v0.6.0 路线图 - 报表分析与数据看板增强

> **状态**：规划中
> **前置**：v0.5.0 投资追踪（✅ 已完成）
> **目标**：让用户从数据中获得洞察，而非仅仅是记录数据

---

## 🎯 核心目标

从「记录工具」升级为「分析工具」—— 用户不仅要看到数据，还要看到趋势、问题、建议。

## 📦 功能模块

### 模块1：月度/年度报表
- 月度收支汇总（收入/支出/结余/储蓄率趋势）
- 年度对比报表（同比、环比）
- 分类支出排行榜（哪个分类花钱最多）
- 导出为 PDF/图片

### 模块2：财务健康评分看板
- 基于5维评估模型（储蓄/投资/债务/收入稳定/成长）
- 动态评分 + 雷达图
- 每个维度的具体建议（如：储蓄率75%→建议提高到80%）

### 模块3：目标进度仪表盘
- 各财务目标的进度条 + 预计完成时间
- 里程碑达成提醒
- 与实际财务数据的联动计算
