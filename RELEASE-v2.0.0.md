# 财富自由之路 v2.0.0

## 🎯 一款帮你实现财务自由的桌面应用

从财务保障到财务安全再到财务自由，三阶段递进式财富管理系统。

## ✨ v2.0.0 亮点

### 🛡️ 全局错误处理体系
- 三阶段统一错误管理：ErrorBoundary → GlobalHandlers → safeCall
- ~40+ 处重复错误处理统一消除，应用稳定性大幅提升

### 🌐 多语言支持 (v1.9.0)
- 轻量级 i18n 系统，无 vue-i18n 依赖
- 中英文双语，全部 29 个视图页面完成适配
- Element Plus locale 联动

### 🔐 数据加密 (v1.9.0)
- AES-256-GCM + PBKDF2 密钥派生
- 本地数据加密存储，隐私安全

### 💱 多币种支持 (v1.9.0)
- 多币种账户管理 + 基准币转换
- Dashboard 多币种格式化展示

### 📊 数据可视化大屏 (v1.7.0)
- ECharts 6 大卡片，3×2 网格深色主题
- 自动轮播 + 截图导出

### 🤖 AI 财务助手 (v1.3.0)
- AI 驱动的财务建议和智能分析
- 功能门控：免费版限制调用次数

### 🔑 订阅授权系统 (v1.0.0)
- 在线许可证验证 + 离线降级
- 三版定价：免费版 / Pro 月度 ¥19 / Pro 终身 ¥399

## 📦 核心功能

| 功能 | 说明 |
|------|------|
| 财务追踪 | 收支记录、账户管理、负债追踪 |
| 投资管理 | 持仓总览、资产配置、收益计算 |
| 预算管理 | 月度预算、分类控制、趋势分析 |
| 目标追踪 | 财务三阶段、进度可视化 |
| 可视化报表 | Dashboard、PDF 导出、Excel 导出 |
| 数据可视化大屏 | ECharts 深色主题大屏展示 |
| 周期性交易 | 自动记录固定收支 |
| 定时备份 | 启动备份 + 每 6 小时自动备份 |
| 数据导入 | 支付宝/微信/通用 CSV 导入 |
| 多语言 | 中英文双语支持 |
| 数据加密 | AES-256-GCM 本地加密 |

## 🖥️ 技术栈

- **桌面端**: Electron + Vue 3 + TypeScript + Element Plus
- **数据层**: SQLite (better-sqlite3)
- **图表**: ECharts + Recharts
- **架构**: pnpm monorepo (desktop + shared + web)
